#### Software Architecture
Built on the Arduino framework under the PlatformIO platform.


#### Installation Tutorial
Use the PlatformIO plugin in VSCode.

#### User Instructions
In `src/main.cpp`, change  
const char *ssid = "********";  
const char *password = "********";  
Change `ssid` to your own WiFi name and `password` to your own WiFi password. Make sure the WiFi supports the 2.4G band.

You can check the "output format" in the table below. If it has "compressed," you can change the camera initialization configuration to config.pixel_format = PIXFORMAT_JPEG; this can increase the frame rate.

After downloading, open the Serial Monitor to check the printed IP address. Enter this IP address in your browser to view the streaming status.
